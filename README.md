## ðŸ’» Code Features
The script `tov_solver.py` implements a rigorous astrophysical solver including:
* **Exact SLy4 Implementation:** Using the piecewise polytrope parametrization from *Read et al. (2009)*.
* **Metric Perturbations:** Solves for the Love number $k_2$ and Tidal Deformability $\Lambda$.
* **Causality Enforcement:** A dynamic Rhoades-Ruffini cutoff to ensure $v_s \le c$ at all densities.
* **Numerical Stability:** Uses PCHIP interpolation and strict monotonicity checks to avoid numerical artifacts.
